## Project Overview
This project presents an interactive Employee Attendance Dashboard built using Power BI. The dashboard helps analyze employee attendance patterns and provides insights into workforce participation using visual analytics.

## Objectives
- Track total enrolled employees
- Analyze present and absent counts
- Calculate average attendance percentage
- Compare attendance performance across teams and managers

## Dashboard Features
- KPI cards showing Total Enrolled, Present, Absent, and Average Attendance
- Team-wise attendance visualization
- Manager-wise attendance comparison
- Interactive and easy-to-understand visual reports

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Insights
The dashboard helps identify attendance trends across different teams and managers, making it easier to understand workforce engagement and attendance performance.

## Learning Outcome
Through this project, I learned how to:
- Create interactive dashboards in Power BI
- Use data visualization to present insights
- Structure business data into meaningful reports
